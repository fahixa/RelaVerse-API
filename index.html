Change Password

URL

/api/users/change-password/:id

Method

PUT

Request Body

    {
        "currentPassword" : "string",
        "newPassword" : "string"
    }

Note

Include authorization token in headers.

Response

    {
        "error": false,
        "message": "Password updated successfully"
    }

        "userId": number,
        "latitude": number,
        "longitude": number,
        "contact": "string",

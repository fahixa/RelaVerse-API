Change Password

URL

/api/users/change-password/:id

Method

PUT

Request Body

    {
        "currentPassword" : "string",
        "newPassword" : "string"
    }

Note

Include authorization token in headers.

Response

    {
        "error": false,
        "message": "Password updated successfully"
    }

Create Campaign

Method

POST

Request Body

        "title": "string",

        "userId": number,
        "latitude": number,
        "longitude": number,
        "contact": "string",
        "description": "string",
        "date": "string",
        "location": "string",
        "whatsappLink": "string",
        "photoEvent": file
    }

        "error": false,
        "message": "Campaign created successfully"
    }
